## 产品概述

一款类似SBTI风格的娱乐性游戏人格测试网页，通过30道趣味选择题分析玩家的游戏人格类型，并推荐适合的游戏（包括皇室战争、荒野乱斗、部落冲突）。

## 核心功能

1. **趣味测试系统**：30道情景选择题，每题3个选项，涵盖游戏态度、社交偏好、竞技心态、时间投入、风格倾向5个维度
2. **人格类型系统**：设计12-15种有趣的游戏人格类型（如：肝帝、佛系玩家、社交达人、竞技狂魔等）
3. **游戏推荐引擎**：根据人格维度分析，推荐包含皇室战争、荒野乱斗、部落冲突在内的游戏清单
4. **结果分享功能**：支持生成结果卡片分享至社交媒体
5. **趣味性设计**：题目和结果描述具有网感、幽默感，吸引玩家参与

## 测试维度设计

1. **游戏态度**：对游戏认真程度/投入方式
2. **社交偏好**：单人游戏vs联机社交
3. **竞技心态**：胜负欲/竞争意识
4. **时间投入**：肝度/氪金意愿
5. **风格倾向**：策略型/操作型/收集型

## 游戏推荐池（部分）

- 皇室战争（策略竞技）
- 荒野乱斗（快节奏竞技）
- 部落冲突（策略养成）
- 原神、王者荣耀、我的世界、蛋仔派对等

## 技术选型

- **前端框架**：纯HTML5 + CSS3 + Vanilla JavaScript（无框架依赖，单文件可运行）
- **样式方案**：内嵌CSS + CSS变量实现主题切换
- **数据存储**：JavaScript对象存储测试题目和人格定义
- **响应式设计**：Flexbox + Grid布局，支持移动端和桌面端

## 架构设计

### 单页应用结构

```
HTML结构：
├── 欢迎页（标题 + 开始测试按钮）
├── 测试区（题目展示 + 选项 + 进度条）
├── 结果页（人格类型 + 游戏推荐 + 分享按钮）
└── 重新测试按钮
```

### 算法逻辑

1. 每个选项对应不同维度的分数加成
2. 5个维度分别计算总分
3. 根据分数分布匹配最接近的人格类型
4. 人格类型对应推荐游戏列表

### 数据结构

```javascript
// 人格类型定义
personalities = {
  "GOD_HAND": {
    name: "操作之神",
    description: "...",
    games: ["荒野乱斗", "皇室战争", "王者荣耀"]
  }

// 题目定义
questions = [
  {
    dimension: "attitude", // 维度
    options: [
      { text: "...", scores: { attitude: 2, social: 0, ... } }
    ]
  }
]
```

## 设计风格

采用赛博朋克+霓虹风格的深色主题，营造游戏氛围感。渐变霓虹色彩搭配暗色背景，营造沉浸式游戏体验。加入微动画和交互反馈，提升趣味感。

## 页面结构

1. **欢迎页**：大标题 + 趣味副标题 + 开始按钮，带动态背景效果
2. **测试页**：题目卡片 + 三个选项按钮 + 底部进度条，选项hover时有霓虹光效
3. **结果页**：人格类型大标题 + 趣味描述 + 推荐游戏卡片网格 + 分享按钮

## 视觉元素

- 霓虹渐变色彩（紫色、青色、粉色）
- 暗色背景 + 玻璃拟态卡片
- 动态光效和微动画
- 游戏风格的图标装饰

## 字体系统

- 标题：思源黑体 Bold / 无衬线粗体
- 正文：系统默认无衬线字体
- 英文游戏名称：等宽字体效果
